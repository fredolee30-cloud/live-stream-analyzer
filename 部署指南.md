# 快速部署指南

## 第 1 步：配置 API Key

用记事本打开 `index.html`，找到第 532 行：

```javascript
const KIMI_API_KEY = 'sk-your-api-key-here';
```

替换为你的 Kimi API Key：

```javascript
const KIMI_API_KEY = 'sk-xxxxxxxxxxxxxxxxxxxxxxxx';
```

**获取 API Key**：https://platform.moonshot.cn/console/api-keys

---

## 第 2 步：部署到 Netlify（2 分钟）

### 方法 A：拖拽部署（无需注册）

1. 打开 https://app.netlify.com/drop
2. 把 `index.html` 拖到虚线框内
3. 等待上传完成
4. 获得链接，例如：`https://random-name-12345.netlify.app`

### 方法 B：自定义链接名字

1. 注册 https://netlify.com
2. 登录后找到你的网站
3. 点击 "Site settings" → "Change site name"
4. 设置为喜欢的名字，例如：
   ```
   https://live-analyzer.netlify.app
   ```

---

## 第 3 步：分享给同事

把部署后的链接发给同事，他们就可以：
1. 打开链接
2. 粘贴直播文稿
3. 点击"开始分析"
4. 查看报告并导出

---

## 自定义域名（可选）

如果需要绑定自己的域名：
1. 在 Netlify 设置中添加自定义域名
2. 按提示配置 DNS 解析

---

## 常见问题

**Q: 部署后显示"请先配置 API Key"？**
A: 确保已修改 `index.html` 中的 `KIMI_API_KEY`，并重新部署。

**Q: 如何更新代码？**
A: 修改 `index.html` 后，重新拖拽到 Netlify Drop。

**Q: 访问速度慢？**
A: 试试 Cloudflare Pages：https://pages.cloudflare.com

---

## 部署到 Vercel（备选）

如果 Netlify 访问慢，可以部署到 Vercel：

1. 运行 `deploy.bat`，推送代码到 GitHub
2. 访问 https://vercel.com
3. 导入仓库，自动部署

---

## 部署成功检查

部署完成后，访问链接应该能看到：
- 顶部标题"📊 直播文稿分析工具"
- "⚙️ 配置"区域（只有模型选择）
- "📝 输入直播文稿"输入框
- 三个按钮："开始分析"、"清空"、"加载示例"
